# n8n-mcp-automation

This repository contains a modular, automated agent built with **n8n**, **LangChain**, and **Google Gemini**, designed to intelligently interact with Gmail and Google Drive.

The goal of this project is to streamline daily workflows by allowing natural language interaction with your Google Workspace ‚Äî enabling tasks like searching emails, summarizing documents, or locating files via smart prompts.

- **Conversational Agent**

  - Built in n8n for easy modification.
